# Customer Churn Analytics Dashboard

## Project Overview

This project is an interactive Customer Churn Analytics Dashboard developed using Power BI. The dashboard provides insights into customer behavior, churn trends, and retention factors, helping businesses identify customers at risk of leaving and make data-driven decisions.

## Objectives

* Analyze customer churn patterns.
* Identify factors influencing customer retention.
* Monitor key business metrics through interactive dashboards.
* Provide actionable insights for improving customer loyalty.

## Tools & Technologies

* Power BI
* Power Query
* DAX
* Microsoft Excel
* Data Visualization

## Dataset Features

The dataset includes:

* Customer ID
* Credit Score
* Country
* Gender
* Age
* Tenure
* Balance
* Number of Products
* Credit Card Status
* Active Member Status
* Estimated Salary
* Churn Status

## Key Performance Indicators (KPIs)

* Total Customers
* Total Churned Customers
* Churn Rate (%)
* Average Credit Score
* Average Balance
* Average Tenure

## Dashboard Features

### Customer Overview

* Churn vs Non-Churn Distribution
* Customer Distribution by Country
* Customer Distribution by Gender

### Churn Analysis

* Churn by Age Group
* Churn Rate by Tenure
* Churn by Product Count
* Active Members vs Churn

### Financial Analysis

* Balance vs Credit Score Analysis
* Churn by Balance Segments

### Interactive Features

* Dynamic Slicers
* Cross Filtering
* Drill-Down Analysis
* Reset Filters Button

## Key Insights

* Customer churn varies across different age groups and countries.
* Active members tend to have lower churn rates.
* Product ownership impacts customer retention.
* Certain balance ranges exhibit higher churn tendencies.
* Tenure plays a significant role in customer loyalty.
