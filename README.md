# Customer Churn Analytics Dashboard

## Overview

This project presents an interactive Customer Churn Analytics Dashboard developed using Power BI. The dashboard helps analyze customer behavior, identify churn patterns, and uncover factors that influence customer retention.

The objective is to provide business stakeholders with actionable insights that can help reduce customer attrition and improve retention strategies.

## Problem Statement

Customer churn is one of the biggest challenges for businesses. Understanding why customers leave and identifying high-risk customer segments can help organizations take proactive measures to improve customer satisfaction and retention.

## Dataset Features

The dataset contains customer information such as:

* Customer ID
* Credit Score
* Country
* Gender
* Age
* Tenure
* Balance
* Number of Products
* Credit Card Status
* Active Member Status
* Estimated Salary
* Churn Status

## Key Insights

* Churn trends vary across age groups and countries.
* Customer activity level significantly impacts retention.
* Product ownership influences churn behavior.
* Customers with specific balance ranges show higher churn risk.
